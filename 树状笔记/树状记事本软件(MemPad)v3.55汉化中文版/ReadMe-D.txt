+++ Installation mit Windows' "Extrakt"-Funktion

Wenn Windows' eingebaute "Extrakt"-Funktion genutzt wird, um die heruntergeladene ZIP-Datei zu entpacken, muss zunächst die ZIP-Datei "Entsperrt" werden (rechte Maustaste, Eigenschaften / Registerkarte: Allgemein, unten). Andernfalls werden alle Dateien aus Gründen der Sicherheit blockiert, was zu Problemen beim Öffnen der Dateien führen kann.

Keine Probleme bei Verwendung anderer ZIP-Tools.


+++ Windows Vista, 7, 8:

Bitte in einen Ordner außerhalb "Program Files" bzw. "Programme" installieren!

Dieses "portable" Programm speichert die Konfigurationsdaten standardmäßig im Programm-Ordner (*), was jedoch innerhalb der "Program Files"-Struktur nicht erlaubt ist. In diesem Fall würde Windows die Dateien in einen "VirtualStore"-Ordner (unter "Benutzer") umleiten.

(*) Es ist auch möglich, eine Daten-Ordner nach Wahl zu verwenden (siehe Hilfe für weitere Informationen).


+++ MemPad ausprobieren:

Für einen Einstieg in die Arbeitsweise von MemPad einfach die Datei "Mempad-d.help" per Drag&Drop auf die "Mempad.exe" ziehen. Zunächst den Abschnitt Programm/Einrichtung lesen.

Damit einige besondere Funktionen aus der Hilfe heraus ausprobiert werden können, sind die Dateien aus dem "Example"-Ordner erforderlich. Bitte diese Dateien ins Hauptverzeichnis kopieren. 


+++ Software-Update:

Zip-Datei in den vorhandenen MemPad-Ordner entpacken. Die Daten bleiben unberührt.
